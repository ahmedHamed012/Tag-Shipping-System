# ⚡ دليل البدء السريع

## إذا كنت مستعجلًا، اتبع هذه الخطوات فقط:

### 1. التثبيت السريع

```bash
npm install
```

### 2. إعداد قاعدة البيانات

```bash
# انسخ الملف
Copy-Item .env.example .env

# عدّل .env وأضف اتصال SQL Server
# مثال: DATABASE_URL="mssql://sa:password@localhost:1433/shipping_system"
```

### 3. إنشاء الجداول

```bash
npx prisma migrate dev --name init
```

### 4. ملء البيانات (اختياري)

```bash
npm run seed
```

### 5. التشغيل

```bash
npm start
```

### 6. افتح المتصفح

```
http://localhost:3000/users
```

## 🎯 ما الذي يمكنك فعله الآن؟

| الميزة         | الرابط                 | الوصف                 |
| -------------- | ---------------------- | --------------------- |
| عرض المستخدمين | `/users`               | جدول بجميع المستخدمين |
| إضافة مستخدم   | زر "إضافة مستخدم جديد" | نموذج في نافذة منبثقة |
| رفع Excel      | زر "رفع من Excel"      | استيراد عدة مستخدمين  |
| قالب Excel     | زر "تحميل قالب Excel"  | تحميل النموذج للملء   |
| تعديل          | كل صف في الجدول        | عدّل بيانات المستخدم  |
| تفعيل/تعطيل    | أيقونة التبديل         | غيّر حالة المستخدم    |
| حذف            | أيقونة السلة           | احذف المستخدم         |

## 🆘 مشاكل شائعة؟

### خطأ "لا يمكن الاتصال بقاعدة البيانات"

- تأكد من تشغيل SQL Server
- تحقق من قيمة DATABASE_URL في .env

### خطأ "الملف لم يُوجد"

```bash
npm install
```

### تريد إعادة تعيين البيانات؟

```bash
npx prisma migrate reset
npm run seed
```
