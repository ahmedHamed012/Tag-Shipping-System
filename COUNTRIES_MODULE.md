# 🌍 وحدة المحافظات والمدن - Countries Module

## نظرة عامة

وحدة متكاملة لإدارة المحافظات والمدن المرتبطة بنظام الشحنات والتجار، مع إمكانية:

- إضافة/تعديل/حذف المحافظات والمدن
- رفع بيانات المحافظات والمدن من ملفات Excel
- استخدام القوائم المنسدلة في نماذج الشحنات والتجار
- حساب سعر الشحنة بناءً على المحافظة والمدينة المختارة

## 📁 البنية والملفات

### 1. **قاعدة البيانات (Prisma)**

```
prisma/schema.prisma
  ├── Governorate Model
  │   ├── id (UUID)
  │   ├── nameAr (String - اسم المحافظة بالعربية)
  │   ├── nameEn (String - اسم المحافظة بالإنجليزية)
  │   ├── isActive (Boolean)
  │   ├── isDeleted (Boolean - soft delete)
  │   ├── createdAt, updatedAt
  │   ├── createdBy, updatedBy (User relations)
  │   └── cities (City[] - المدن التابعة)
  │
  └── City Model
      ├── id (UUID)
      ├── nameAr (String - اسم المدينة بالعربية)
      ├── nameEn (String - اسم المدينة بالإنجليزية)
      ├── governorateId (Foreign Key)
      ├── isActive (Boolean)
      ├── isDeleted (Boolean - soft delete)
      ├── createdAt, updatedAt
      ├── createdBy, updatedBy (User relations)
      └── governorate (Governorate relation)
```

### 2. **معالج البيانات (Controller)**

```
controllers/countries.controller.js
  ├── getAllGovernorates() - عرض جميع المحافظات مع الترقيم والبحث
  ├── getCitiesByGovernorate() - الحصول على مدن محافظة معينة
  ├── getAllGovernoratesForDropdown() - الحصول على المحافظات للقوائم المنسدلة
  ├── createGovernorate() - إنشاء محافظة جديدة
  ├── updateGovernorate() - تعديل بيانات المحافظة
  ├── deleteGovernorate() - حذف محافظة
  ├── createCity() - إنشاء مدينة جديدة
  ├── updateCity() - تعديل بيانات المدينة
  ├── deleteCity() - حذف مدينة
  └── bulkUploadCountries() - رفع بيانات من ملف Excel
```

### 3. **مسارات API (Routes)**

```
routes/countries.routes.js
  ├── GET /countries - عرض الصفحة الرئيسية
  ├── POST /countries/api/create-governorate - إنشاء محافظة
  ├── POST /countries/api/update-governorate/:id - تعديل محافظة
  ├── DELETE /countries/api/delete-governorate/:id - حذف محافظة
  ├── POST /countries/api/create-city - إنشاء مدينة
  ├── POST /countries/api/update-city/:id - تعديل مدينة
  ├── DELETE /countries/api/delete-city/:id - حذف مدينة
  ├── GET /countries/api/cities/:governorateId - الحصول على مدن محافظة
  ├── GET /countries/api/governorates-dropdown - المحافظات للقوائم المنسدلة
  ├── POST /countries/api/bulk-upload - رفع ملف Excel
  └── GET /countries/api/download-template - تحميل قالب Excel
```

### 4. **واجهات المستخدم (Views)**

```
views/countries/
  ├── index.pug - الصفحة الرئيسية مع جدول المحافظات
  ├── _governorateModal.pug - نموذج إضافة/تعديل محافظة
  ├── _cityModal.pug - نموذج إضافة/تعديل مدينة
  └── _uploadModal.pug - نموذج رفع ملف Excel
```

### 5. **أدوات مساعدة (Utilities)**

```
utils/excelUtils.js
```

## 🚀 الاستخدام

### 1. **الوصول إلى الوحدة**

```
http://localhost:3000/countries
```

### 2. **إضافة محافظة**

- اضغط على زر "إضافة محافظة"
- أدخل الاسم بالعربية والإنجليزية (اختياري)
- اضغط "حفظ"

### 3. **إضافة مدينة**

- في جدول المحافظات، اضغط على زر "المدن" (أيقونة المبنى)
- ستظهر جميع المدن الموجودة
- اضغط "إضافة مدينة"
- أدخل بيانات المدينة
- اضغط "حفظ"

### 4. **رفع ملف Excel**

- اضغط على زر "رفع ملف"
- اختر ملف Excel يحتوي على:
  - **العمود الأول**: اسم المحافظة (عربي)
  - **العمود الثاني**: اسم المحافظة (إنجليزي)
  - **العمود الثالث**: اسم المدينة (عربي)
  - **العمود الرابع**: اسم المدينة (إنجليزي)
- اضغط "رفع"
- سيتم عرض تقرير بنتائج الرفع

### 5. **استخدام في الشحنات**

عند إنشاء شحنة جديدة:

- سيظهر حقل "المحافظة" كقائمة منسدلة
- اختر المحافظة
- سيتم تحميل المدن تلقائياً
- اختر المدينة المناسبة

### 6. **استخدام في قائمة أسعار التاجر**

عند إضافة بند في قائمة الأسعار:

- اختر المحافظة من القائمة المنسدلة
- سيتم تحميل المدن التابعة
- اختر المدينة
- أدخل السعر

## 🔄 التكامل مع نظام السعر

### عملية حساب سعر الشحنة:

1. **اختيار المحافظة والمدينة**

   ```javascript
   // يتم اختيار المحافظة والمدينة من القوائم المنسدلة
   // مثال: الرياض - حي النخيل
   ```

2. **البحث عن قائمة الأسعار**

   ```sql
   SELECT price FROM merchant_prices
   WHERE merchantId = :merchantId
   AND governorate = 'الرياض'
   AND city = 'حي النخيل'
   AND isDeleted = false
   ```

3. **حساب السعر النهائي**
   - السعر الأساسي من قائمة الأسعار
   - إضافة رسوم التوصيل السريع (إن وجدت) = السعر × 0.5
   - إضافة رسوم الوزن الزائد = (الوزن الفعلي - الحد الأقصى) × رسم الكيلو الواحد

## 🛠️ استكشاف الأخطاء

### المشكلة: لا تظهر المحافظات في القائمة المنسدلة

**الحل:**

1. تأكد من تشغيل الخادم
2. تحقق من وجود بيانات في جدول `governorates`
3. تحقق من الكونسول من وجود أخطاء في الشبكة

### المشكلة: لا تظهر المدن بعد اختيار محافظة

**الحل:**

1. تأكد من أن المحافظة تحتوي على مدن
2. أضف مدن جديدة للمحافظة
3. تحقق من رابط API `/countries/api/cities/:governorateId`

### المشكلة: فشل رفع ملف Excel

**الحل:**

1. تحقق من صيغة الملف (.xlsx أو .xls)
2. تأكد من وجود البيانات بدءاً من الصف الثاني
3. تأكد من ملء العمودين الأول والثالث على الأقل

## 📝 ملاحظات مهمة

1. **الحذف الآمن (Soft Delete)**
   - لا يتم حذف البيانات فعلياً من قاعدة البيانات
   - يتم وضع علامة `isDeleted = true` فقط
   - يمكن استرجاع البيانات المحذوفة من قاعدة البيانات إذا لزم الأمر

2. **تتبع المستخدمين**
   - يتم تسجيل من قام بإنشاء/تعديل كل بيانة
   - يتم حفظ تاريخ الإنشاء والتعديل

3. **الترتيب**
   - المحافظات مرتبة حسب تاريخ الإنشاء (الأحدث أولاً)
   - المدن مرتبة أبجدياً حسب الاسم

4. **التوفر والتفعيل**
   - يمكن تفعيل/تعطيل المحافظات والمدن (الحقل `isActive`)
   - المحافظات المعطلة لن تظهر في القوائم المنسدلة
